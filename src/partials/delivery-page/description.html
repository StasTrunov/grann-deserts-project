Опис Умови зберігання Доставка

Ніжний десерт, який знайомий усім нам з дитинства.

Ви маєте можливість замовити класичний торт без

прикрас, або ж прикрасити його соковитими

ягодами або карамеллю.

Не відмовляйте собі у спокусі скуштувати “Наполеон”

! Ви маєте можливість замовити торт без цукру, без

глютену та без лактози.
